# L&M's Playhall

L&M's Playhall - An educational game to help children learn the alphabet!

## Files in this project:
- `index.html` - Main HTML file
- `game.js` - Game code
- `README.md` - This file

## How to run:

### Method 1: Direct (easiest)
1. Double-click `index.html`
2. Game should open in your browser

### Method 2: With local server (if Method 1 doesn't work)

**Windows:**
```
python -m http.server 8080
```

**Mac/Linux:**
```
python3 -m http.server 8080
```

Then open your browser to: http://localhost:8080

## How to play:
1. A letter will appear at the top (e.g., "Find: A")
2. Letters fall from the sky
3. Click the correct letter!
4. Score points for each correct letter

## What your child will learn:
- Letter recognition (A-Z)
- Letter names
- Hand-eye coordination

## Customization ideas:
- Adjust letter fall speed
- Add sounds/voice
- Change colors
- Add different difficulty levels
- Practice only certain letters (A-F for beginners)

Enjoy playing!
