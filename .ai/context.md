# L&M's Playhall - Project Context

## Project Vision

L&M's Playhall is a collection of small, educational browser-based games designed to help children (and adults) learn, focus, and relax. Each game is crafted with specific educational or wellness goals in mind.

## Target Audience

- **Primary**: Children ages 3-10 learning foundational skills
- **Secondary**: Parents and educators looking for quality educational tools
- **Tertiary**: Adults seeking focus and relaxation exercises

## Core Principles

1. **Simplicity**: Each game should be intuitive and require minimal explanation
2. **Educational Value**: Every game teaches or reinforces a specific skill
3. **Engagement**: Games should be fun and visually appealing
4. **Accessibility**: Browser-based, no installation required, mobile-friendly
5. **Privacy**: No data collection, no accounts, no tracking
6. **Modularity**: Each game is independent and self-contained

## Technology Stack

- **Framework**: Kaboom.js (game engine)
- **Language**: Vanilla JavaScript (ES6+)
- **Testing**: Puppeteer (headless browser testing)
- **Hosting**: GitHub Pages
- **Deployment**: Git + GitHub CLI

## Success Metrics

- Easy to add new games (< 1 day per simple game)
- Each game loads quickly (< 3 seconds)
- Games work on mobile and desktop
- Code is maintainable and well-documented
- AI assistants can easily understand and extend the codebase
